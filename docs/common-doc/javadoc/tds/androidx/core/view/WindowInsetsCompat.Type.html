      • systemGestures

         static int systemGestures()

        Returns an insets type representing the system gesture insets.

        The system gesture insets represent the area of a window where system gestures havepriority and may consume some or all touch input, e.g. due to the a system baroccupying it, or it being reserved for touch-only gestures.

        Simple taps are guaranteed to reach the window even within the system gesture insets,as long as they are outside the system window insets.

        When SYSTEM_UI_FLAG_LAYOUT_STABLE is requested, an inset will be returnedeven when the system gestures are inactive due to SYSTEM_UI_FLAG_LAYOUT_FULLSCREEN or SYSTEM_UI_FLAG_LAYOUT_HIDE_NAVIGATION.
