abstract fun areItemsTheSame(@NonNull() oldItem: T, @NonNull() newItem: T): Boolean
Content copied to clipboard

Called to check whether two objects represent the same item.

For example, if your items have unique ids, this method should check their id equality.

Note: {@code null} items in the list are assumed to be the same as another {@code null} item and are assumed to not be the same as a non-{@code null} item. This callback willnot be invoked for either of those cases.

Return

True if the two items represent the same object or false if they are different.
