FLAG_REPORT_VIEW_IDS

val FLAG_REPORT_VIEW_IDS: Int
Content copied to clipboard

This flag requests that the AccessibilityNodeInfos obtainedby an AccessibilityService contain the id of the source view.The source view id will be a fully qualified resource name of theform "package:id/name", for example "foo.bar:id/my_list", and it isuseful for UI test automation. This flag is not set by default.
