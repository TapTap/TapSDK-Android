1010 indicates that an endpoint (client) is terminating the connection because it has expectedthe server to negotiate one or more extension, but the server didn't return them in theresponse message of the WebSocket handshake. The list of extensions which are needed SHOULDappear in the /reason/ part of the Close frame. Note that this status code is not used by theserver, because it can fail the WebSocket handshake instead.
