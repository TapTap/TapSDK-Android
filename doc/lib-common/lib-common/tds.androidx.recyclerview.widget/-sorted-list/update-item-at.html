updateItemAt

open fun updateItemAt(index: Int, item: T)
Content copied to clipboard

Updates the item at the given index and calls onChanged and/or onMoved if necessary.

You can use this method if you need to change an existing Item such that its position in thelist may change.

If the new object is a different object (get(index) != item) and areContentsTheSame returns true, SortedListavoids calling onChanged otherwise it calls onChanged.

If the new position of the item is different than the provided index,SortedListcalls onMoved.
