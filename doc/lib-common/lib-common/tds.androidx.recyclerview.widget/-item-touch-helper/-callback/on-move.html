onMove

abstract fun onMove(@NonNull() recyclerView: RecyclerView, @NonNull() viewHolder: RecyclerView.ViewHolder, @NonNull() target: RecyclerView.ViewHolder): Boolean
Content copied to clipboard

Called when ItemTouchHelper wants to move the dragged item from its old position tothe new position.

If this method returns true, ItemTouchHelper assumes {@code viewHolder} has been movedto the adapter position of {@code target} ViewHolder( ViewHolder#getAdapterPositionInRecyclerView()).

If you don't support drag & drop, this method will never be called.

Return

True if the {@code viewHolder} has been moved to the adapter position of {@code target}.
