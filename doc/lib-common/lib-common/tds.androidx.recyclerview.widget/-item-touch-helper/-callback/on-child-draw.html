Called by ItemTouchHelper on RecyclerView's onDraw callback.

If you would like to customize how your View's respond to user interactions, this isa good place to override.

Default implementation translates the child by the given dX,dY.ItemTouchHelper also takes care of drawing the child after other children if it is beingdragged. This is done using child re-ordering mechanism. On platforms prior to L, thisisachieved via getChildDrawingOrder and on Land after, it changes View's elevation value to be greater than all other children.)
