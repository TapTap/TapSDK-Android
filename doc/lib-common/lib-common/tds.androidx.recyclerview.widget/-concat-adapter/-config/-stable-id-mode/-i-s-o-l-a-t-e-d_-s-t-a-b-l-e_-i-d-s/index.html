In this mode, ConcatAdapter will return {@code true} from hasStableIds and will require all added Adapters to have stable ids. As two different adapters may returnsame stable ids because they are unaware of each-other, ConcatAdapter willisolate each Adapter's id pool from each other such that it will overwritethe reported stable id before reporting back to the RecyclerView. In thismode, the value returned from getItemId might differ from thevalue returned from getItemId.Adding an adapter without stable ids will result in an IllegalArgumentException.
