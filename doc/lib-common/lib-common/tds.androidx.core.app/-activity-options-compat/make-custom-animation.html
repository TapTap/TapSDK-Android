Create an ActivityOptions specifying a custom animation to run when theactivity is displayed.

Return

Returns a new ActivityOptions object that you can use to supplythese options as the options Bundle when starting an activity.

Parameters

context

Who is defining this. This is the application that theanimation resources will be loaded from.

enterResId

A resource ID of the animation resource to use for theincoming activity. Use 0 for no animation.

exitResId

A resource ID of the animation resource to use for theoutgoing activity. Use 0 for no animation.
